Tank Wars Mobile

Classic artillery game rebuilt for mobile — 2-player local and online.

Overview

Tank Wars Mobile is a faithful recreation of the classic DOS artillery game Tank Wars 3.2 (1992) by Kenneth Morse, rebuilt from the ground up for mobile touchscreens. Two players take turns aiming their tanks and firing across randomly generated destructible terrain. Judge the angle, power, and wind to land your shots — there's no trajectory preview, so every shot is a test of skill and intuition.

Built as a Vite + Capacitor app; online multiplayer is brokered through Firebase.

Features

  • 16 weapons: lead balls, hand grenades, incinerators, 20 K nukes, 5-megaton nukes, MIRVs, laser blasters, chain reaction inducers (4 variants), sonic blasters, dirt balls, explosive dirt
  • 3 defence systems: energy shields, inertia dampeners, repulsers
  • 6 wall types: none, sticky, elastic, accelerating, warping, random
  • 5 wind levels plus random — from calm to gale-force
  • Destructible terrain with configurable crumbling (0–100 %)
  • Weapon shop between rounds — earn from kills, spend on firepower
  • Synthesised retro sound effects
  • Local 2-player hot-seat on one device
  • Online multiplayer via Firebase (room code or quick match)
  • Touch controls: drag to aim, tap to fire

Build & install

npm install
npm run build         # vite build into dist/
npx cap sync android
cd android && ./gradlew assembleRelease

# Optional: run the relay server
npm run server
